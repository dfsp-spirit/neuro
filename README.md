## Building and running the demo app


### Building manually

Build:

```shell
go build cmd/example1/example_neurogo.go
```

Then run it:

```shell
./example_neurogo --meshfile data/lh.white
```

### If you have `make`

To build and run, use:

```shell
make run
```
